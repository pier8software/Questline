## Classic Text Adventures

### Zork (1977-1979)
- Infocom's flagship text adventure
- Established verb-noun command pattern
- Famous for "You are likely to be eaten by a grue"
- https://en.wikipedia.org/wiki/Zork

### Colossal Cave Adventure (1976)
- The original text adventure by Will Crowther
- Inspired by Mammoth Cave in Kentucky
- Two-word command parser
- https://en.wikipedia.org/wiki/Colossal_Cave_Adventure

## Game Loop Patterns

### REPL Pattern
- Read-Evaluate-Print Loop
- Standard pattern for interactive interpreters
- Applies directly to text adventures

### Command Pattern (GoF)
- Encapsulate requests as objects
- Supports undo, logging, queuing
- Good fit for adventure game commands
